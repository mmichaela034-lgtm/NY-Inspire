# NYS Portrait of a Graduate Chatbot

An AI-powered chatbot that answers questions about the
New York State Portrait of a Graduate framework.

## About
This chatbot uses Anthropic's Claude AI to help educators,
students, and families understand the NYS Portrait of a
Graduate — the new framework replacing Regents exam
requirements by the 2027-28 school year.

## The Six Competencies
1. Critical Thinker
2. Effective Communicator
3. Innovative Problem Solver
4. Literate Across Content Areas
5. Culturally and Socially-Emotionally Competent
6. Global Citizen
